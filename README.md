# Portfolio Project

A modern portfolio application built with industry-standard web development practices.

## 🚀 Project Overview

This project started as a simple static HTML/CSS portfolio website and is being rebuilt into a modern full-stack web application.

The goal is not only to showcase personal projects and experiences, but also to demonstrate software engineering practices commonly used in professional environments.

## 🛠 Tech Stack

### Frontend

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router

### Development Tools

- Git
- GitHub
- pnpm Workspace (Monorepo)
- ESLint

### Planned Stack

- Express.js
- PostgreSQL
- Prisma ORM
- Docker
- GitHub Actions
- Testing (Vitest, React Testing Library, Supertest)
